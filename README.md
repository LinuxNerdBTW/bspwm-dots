### Installation

~~~bash

https://github.com/mmsaeed509/bspwm-dots.git
cd bspwm-dots

# Copy misc to home #
cp misc/* ~/

# Copy scripts to /bin #
sudo cp bin/* /bin

# Copy config to ~/.config #
cp -r config/* ~/.config

# Clean #
cd .. && rm -rf bspwm-dots
~~~
